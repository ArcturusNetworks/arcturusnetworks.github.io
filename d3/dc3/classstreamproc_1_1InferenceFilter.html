Detailed Description

Filter node performing object detection on video frames.

This class uses amv's deepvision library to create neural networks for object detection. The object detection net can be configured by passing special keys which specify parameters such as the object detection model or backend.

See also
streamproc::dvconfig::ObjdetModelMap, streamproc::dvconfig::BackendMap, streamproc::dvconfig::LabelsMap
json/inference.schema.json, json/stream.schema.json, json/address.schema.json

The following sample code demonstrates how to run an InferenceFilter:

namespace sp = streamproc;
// Configure the filter
json config = R"(
{
// Neural net specification
"net" : {
"model" : "GENERAL_SSD",
"modelFile" : "../data/models/ssd_mobilenet_v3_small_coco_2020_01_14.tflite",
"backend" : "TFLITE"
},
// Filter inputs
"incomingStreams" : {
"frame" : {
"address" : {
"ip" : "127.0.0.1",
"port" : 5555
},
"identity" : "InferenceFrame"
}
},
// Filter outputs
"outgoingStreams" : {
"detections" : {
"address" : 5556,
"identity" : "InferenceDetections",
"connections" : [
"SinkDetections"
]
}
}
}
)"_json;
auto inferenceFilter = sp::InferenceFilter::Create(config);
inferenceFilter->Start();
inferenceFilter->Join();
