Detailed Description

Configuration options for neural networks used by different Filter nodes.

Filter nodes such as streamproc::InferenceFilter::InferenceFilter and streamproc::EmbeddingFilter::EmbeddingFilter use neural networks to perform some type of filtering. These neural networks are created using amv's deepvision library. This namespace specifies different configuration values used by the deepvision library which can be passed to Filter nodes when they create their nets.
