Detailed Description

HTTP client which communicates with the InfluxDb HTTP server.

Read operations communicate directly with the database. Write operations are offloaded to a batch proxy which writes points in batches to the database.

See also
streamproc::influxdb::Point, streamproc::influxdb::BatchProxy
namespace sp = streamproc;
sp::influxdb::Client influxDb;
std::vector<sp::influxdb::Point> points;
influxDb.Read("select * from node", points);
influxDb.Write(
sp::influxdb::Point("node")
.Tag("name", "InferenceFilter")
.Field("fps", 10.0)
);
