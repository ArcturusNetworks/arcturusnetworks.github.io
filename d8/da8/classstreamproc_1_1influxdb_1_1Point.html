Detailed Description

Object which represents an influxdb point.

See also
InfluxDb Key Concepts

InfluxDb operates with time series data. A time series is a collection of points that share certain properties. A point represents a single data record that has four components: measurement, tags, fields, and a timestamp. A point is uniquely identified by its series and its timestamp. Points in a series have the same measurement, tags, and fields.

This is the data structure used by the influxdb::Client to represent data sent to the database and data received from the database.

See also

namespace sp = streamproc;
// Create a point
auto p = sp::influxdb::Point("node")
.Tag("name", "VideoSource")
.Field("source", "rtsp://172.0.0.1")
.Tag("machine", "venus")
.Field("fps", 25)
.Field("cpu", 0.879256)
.Field("active", true)
.Time(1586472382550);
