Detailed Description

Base clase for stream.

Interface to allow the transmission of data between nodes in a network. Streams use sockets to process messages sent to or received from a node. Messages are buffered until the stream worker routine receives the message from the buffer and transmits it to other nodes. This enables asynchronous processing of data.

See also
streamproc::models, zmq guide

const std::string streamproc::Stream::identity

Identity of the stream.

Used for logging. Serves as an endpoint address for internal sockets. For incoming streams this is also used as the socket identity for the external stream socket. Outgoing streams use the socket identity to synchronize outputs to multiple incoming streams.
