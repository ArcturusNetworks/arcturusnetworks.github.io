Detailed Description

Interface for asynchronous communication between other nodes in a distributed network.

Each node uses a bus to communicate with every other node in a distributed process. The bus connects to a manager which forwards messages from one node to every other node in a network. A bus consists of a pair of zmq pub/sub sockets which connect to a central proxy controlled by the manager.

See also
streamproc::Manager, zmq guide

The bus also has an interface to communicate with influxdb via an HTTP client.

See also
streamproc::influxdb::Client
