Detailed Description

Source node which generates video frames.

This class uses OpenCV cv::VideoCapture to capture frames from a video and forwards them to other nodes in a pipeline.

See also
json/videosource.schema.json, json/stream.schema.json, json/address.schema.json

The following sample code demonstrates how to run a VideoSource

namespace sp = streamproc;
// Configure the source
json config = R"(
{
"capture" : "../data/videos/AVSS_AB_Easy_Divx.mov",
"outgoingStreams" : {
"frame" : {
"address" : 5555,
"identity" : "SourceFrame",
"connections" : [
"SinkFrame"
]
}
}
}
)"_json;
auto videoSource = sp::VideoSource::Create(config);
videoSource->Start();
videoSource->Join();
