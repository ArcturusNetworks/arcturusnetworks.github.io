Detailed Description

The Python equivalent of streamproc::Filter.

This will allow abstract away a few more boilerplate stuff like constructors so that the developers can focus solely on Process(). All they need to do is to create their own Inference class and define that one function.

Initialises the Inference object.

Following its C++ counterpart, this function is "overloaded".

Parameters
config(dict) the whole Filter config as json
name(str) name of the node/module
incoming(dict) incoming stream config as json
outgoing(dict) outgoing stream config as json

Either use config or the other three explicit parameters to initialize the stream in the inherited class like

super().__init__(config=json_dict)
# or
super().__init__(name='name', incoming=inc, outgoing=out)
