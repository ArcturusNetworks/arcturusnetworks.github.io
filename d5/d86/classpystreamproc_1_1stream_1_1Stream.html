Detailed Description

The Python equivalent of streamproc::Stream.

At the moment the streams don't share a common logger with the Node itself so the logger, subsequently its logging level, is specified separately.

◆ ProcessInternal()

def pystreamproc.stream.Stream.ProcessInternal (   self)

"Virtual" internal process function that needs to be defined.

See also
ProcessExternal

For example, an incoming stream would want to read the internal buffer to fetch new data while an outgoing stream would want to write to the internal buffer so that they will be sent away by the external process.
