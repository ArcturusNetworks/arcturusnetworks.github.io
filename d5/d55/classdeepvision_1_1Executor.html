Detailed Description

Implementations of this interface are responsible for reading from file, optimizing, loading and executing one neural network using a supported backend.

Currently the supported backends are ArmNN and Tensorflow Lite. Implementations make no assumption about the task to be performed, e.g. computer vision. They only deal with input and output tensors, handling much of the boilerplate of the respective backend libraries.
