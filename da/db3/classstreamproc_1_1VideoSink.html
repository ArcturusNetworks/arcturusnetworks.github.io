Detailed Description

Sink node which consumes video frames.

The sink receives video frames and writes them to one or more outputs.

See also
streamproc::videowriter
json/videosink.schema.json, json/stream.schema.json, json/address.schema.json

The following sample code demonstrates how to run an EmbeddingFilter

namespace sp = streamproc;
// Configure the filter
json config = R"(
{
"writers" : [
{
"type" : "videofile",
"filename" : "test.avi",
"width" : 720,
"height" : 576,
"fps" : 24
}
],
"incomingStreams" : {
"frame" : {
"address" : {
"ip" : "127.0.0.1",
"port" : 5555
},
"identity" : "SinkFrame"
}
}
}
)"_json;
auto videoSink = sp::VideoSink::Create(config);
videoSink->Start();
videoSink->Join();
