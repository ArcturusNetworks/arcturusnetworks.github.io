Detailed Description

Since the pystreamproc is mainly used for filter development, these fbs abstraction classes are not as "complete" as the C++ counterparts.

Hence only Encode is implemented here.

This "submodule" contains all the abstractions a Filter node needs to serialize and deserialize its flatbuffers messages. For developers, they may generate "header" files using flatc based on their own flatbuffers schema and write encode/decode functions in their executable. For example, if the schema is

table Box {
TLX: int;
TLY: int;
BRX: int;
BRY: int;
}
table Frame {
rows: int;
cols: int;
img: [ubyte];
}

Then the corresponding abstraction classes can look like

# the generated fbs file (Box.py and Frame.py in this case) can be placed
# in the same directory as the executable
import Box as fbsBox
import Frame as fbsFrame
import flatbuffers
class Box:
def __init__(self, top_left: list, bottom_right: list):
self.top_left = top_left
self.bottom_right = bottom_right
# the encode function must return a byte-like object (like bytes)
def Encode(self) -> bytes:
# initialize the fbs builder
builder = flatbuffers.Builder(0)
# add the attributes specified in the schema
fbsBox.BoxStart(builder)
fbsBox.BoxAddTLX(builder, self.top_left[0])
fbsBox.BoxAddTLY(builder, self.top_left[1])
fbsBox.BoxAddBRX(builder, self.bottom_right[0])
fbsBox.BoxAddBRY(builder, self.bottom_right[1])
box = fbsBox.BoxEnd(builder)
# serialize and return
builder.Finish(box)
return builder.Output()
class Frame:
# the argument must be a byte-like object for the decode function
def Decode(msg: bytes):
# creates a Frame object based on the msg buffer
frame = fbsFrame.Frame.GetRootAsFrame(msg, 0)
# get the image matrix from the fbs object
img = frame.ImgAsNumpy()
# "inflate" the 1-dimensional array back to a matrix
img = img.reshape(frame.Rows(), frame.Cols(), 3)
# then either return the image for simplicity
return img
# or return a whole object for convenient data access
ret = Frame()
ret.img = img
return ret
