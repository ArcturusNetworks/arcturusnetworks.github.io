◆ Parser

enum deepvision::armnn::Parser
strong

Callers should use this enum to indicate which ArmNN parser they'd like to use.

Currently Executor only supports TfLiteParser. It's trivial to support the others in Executor::ParseNetwork, but only TfLiteParser works (well) at the moment.

◆ GetRuntime()

::armnn::IRuntimePtr& deepvision::armnn::GetRuntime ( )

Obtain a reference to a singleton ::armnn::IRuntimePtr.

It should be used to load and run all ArmNN networks. Theoretically ArmNN made the object thread-safe, but in practice, enqueueing workloads from multiple threads may not be desirable since ArmCompute's work scheduler has known multi-threading issues:

https://github.com/ARM-software/armnn/issues/262 https://github.com/ARM-software/ComputeLibrary/issues/742
