Detailed Description

Interface to send data to nodes in a network.

Data is encoded using google flatbuffers and transmitted using zmq sockets. The external socket of the outgoing stream monitors for requests for data from different incoming streams.

See also
streamproc::models, zmq guide, flatbuffers
